## ما تم تنفيذه

### الموقع الرئيسي (Public Website)
- صفحة رئيسية متكاملة بـ 9 أقسام ديناميكية (Hero Slider, منتجات, حلول, عملاء, شهادات, معارض, أخبار, تواصل)
- كتالوج منتجات كامل مع صفحات تفصيلية لكل منتج (معرض صور, مواصفات, منتجات مشابهة)
- صفحات: من نحن, الحلول, العملاء, الشهادات, المعارض, الأخبار, فريق العمل, تواصل معنا
- نموذج تواصل متكامل مع Formspree
- دعم كامل لـ 3 لغات: **العربية** (RTL) | **الإنجليزية** | **التركية**
- تصميم متجاوب (Desktop + Tablet + Mobile)
- تحسين محركات البحث (SEO) مع Meta Tags
- رسوم متحركة احترافية (Framer Motion)
- صفحات سياسة الخصوصية وشروط الاستخدام

### لوحة التحكم (Admin Dashboard)
- إدارة **المنتجات** (إضافة, تعديل, حذف, صور متعددة, SEO)
- إدارة **التصنيفات** (تصنيفات رئيسية وفرعية)
- إدارة **الحلول** الصناعية
- إدارة **الأخبار والمقالات** (محرر محتوى, وسوم, مقالات مميزة)
- إدارة **سلايدر الصفحة الرئيسية** (صور + فيديو)
- إدارة **فريق العمل** (صور, مناصب, روابط LinkedIn)
- إدارة **العملاء والشركاء** (شعارات, مواقع)
- إدارة **الشهادات** (تواريخ إصدار وانتهاء)
- إدارة **المعارض** (تواريخ, مواقع, صور)
- إدارة **المقابلات التلفزيونية**
- إدارة **الكتالوجات** (رفع PDF, تتبع التحميلات)
- **صندوق الرسائل** (رسائل العملاء, قراءة/أرشفة/رد)
- **سجل النشاطات** (تتبع جميع عمليات الإدارة)
- **سلة المحذوفات** (استرجاع العناصر المحذوفة)
- **الإعدادات العامة** (معلومات الموقع, التواصل, السوشيال ميديا, SEO, Analytics)

### البنية التقنية
- **Next.js 15** — أحدث إصدار مع App Router
- **TypeScript** — كود آمن ومنظم
- **Tailwind CSS** — تصميم سريع ومتجاوب
- **Prisma + MySQL** — قاعدة بيانات قوية
- **NextAuth.js** — نظام تسجيل دخول آمن
- **Cloudinary** — استضافة وتحسين الصور
- **PM2** — إدارة العمليات على السيرفر
- **GitHub Actions** — نشر تلقائي عند الدفع لـ master

## كيفية تحديث المحتوى

> لا تحتاج أي خبرة برمجية — كل شيء يتم من لوحة التحكم

1. ادخل على [www.snalattal.me/admin](https://www.snalattal.me/admin)
2. سجّل الدخول بحساب الأدمن
3. اختر القسم المطلوب من القائمة الجانبية
4. أضف / عدّل / احذف المحتوى
5. التغييرات تظهر على الموقع فوراً

---

## كيفية النشر (Deploy)

عند إجراء تعديلات برمجية:

```
git push origin master
```
> يتم النشر تلقائياً عبر GitHub Actions على سيرفر Hostinger
